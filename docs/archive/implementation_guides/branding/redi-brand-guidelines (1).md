## Brand Overview

### Mission
The Resuscitation EDucation Initiative (REdI) delivers centrally coordinated, multidisciplinary education programs for resuscitation and clinical deterioration response across Metro North Health.

### Brand Personality
- **Professional** – Clinically rigorous and evidence-based
- **Approachable** – Welcoming to all disciplines and experience levels
- **Dynamic** – Ready to respond and adapt
- **Collaborative** – Emphasizing interprofessional teamwork
- **Supportive** – Building confidence and capability

### Brand Voice
- Clear and direct
- Encouraging without being patronizing
- Clinically accurate but accessible
- Action-oriented

## Logo System

### Primary Logo

The REdI logo consists of a stylized wordmark where:
- **"R"** – Bold black letterform
- **"Ed"** – Coral/salmon pink with the "E" featuring a distinctive design element (stylized to suggest education/action)
- **"I"** – Bold black letterform

**Tagline:** "Resuscitation EDucation Initiative"

### Logo Variations

| Variation | Usage |
|-----------|-------|
| Full logo with tagline | Primary use on documents, presentations, signage |
| Wordmark only | Limited space applications, digital avatars |
| Stacked version | Square format requirements |
| Reversed (white) | Dark backgrounds |
| Single color (black) | Fax, low-quality printing |

### Clear Space

Maintain minimum clear space equal to the height of the "E" around all sides of the logo.

### Minimum Size

- **Print:** 25mm width minimum
- **Digital:** 100px width minimum

### Logo Don'ts

- ❌ Do not stretch or distort
- ❌ Do not rotate
- ❌ Do not change colors outside approved palette
- ❌ Do not add effects (shadows, gradients, outlines)
- ❌ Do not place on busy backgrounds without contrast panel
- ❌ Do not recreate or approximate the logo

## Typography

### Primary Typeface: Montserrat

A clean, modern sans-serif that balances professionalism with approachability.

| Style | Weight | Usage |
|-------|--------|-------|
| **Montserrat Bold** | 700 | H1 Headers, Logo tagline |
| **Montserrat SemiBold** | 600 | H2, H3 Headers, Buttons |
| **Montserrat Medium** | 500 | H4 Headers, Navigation |
| **Montserrat Regular** | 400 | Body text, paragraphs |

### Display Typeface: Bebas Neue

Bold condensed sans-serif for impact headlines and event branding.

| Style | Usage |
|-------|-------|
| **Bebas Neue Regular** | Event titles, promotional headlines |

### System Fallbacks

```css
/* Primary */
font-family: 'Montserrat', 'Segoe UI', 'Roboto', 'Helvetica Neue', sans-serif;

/* Display */
font-family: 'Bebas Neue', 'Impact', 'Arial Black', sans-serif;
```

## Mascot Guidelines

### The REdI Panda

The REdI mascot is a friendly red panda character wearing clinical scrubs, representing the approachable yet professional nature of the initiative.

### Character Specifications

- **Species:** Red panda (Ailurus fulgens)
- **Attire:** Black/charcoal clinical scrubs with REdI branding
- **Personality:** Friendly, helpful, ready to teach
- **Pose variations:** Waving, presenting, demonstrating

### Usage Guidelines

**Do:**
- Use official mascot artwork only
- Maintain original proportions
- Use on welcoming/introductory materials
- Use for event promotion and wayfinding

**Don't:**
- Modify the character design
- Add accessories not in the original design
- Use in clinical documentation requiring formality
- Place in contexts that diminish professionalism

### Mascot Placement

- Event promotional materials
- Social media content
- Training welcome screens
- Wayfinding signage
- Merchandise

## Sub-Brand System

REdI encompasses several training programs, each with distinct visual identity while maintaining brand cohesion.

### Program Identities

#### ALS (Advanced Life Support - Adult)
- **Primary Color:** REdI Navy `#1B3A5F`
- **Accent Color:** Gold/Yellow circles in logo
- **Logo Style:** Stylized "ALS" with circular elements

#### ReACT (Response to Acute Crisis Training)
- **Primary Color:** REdI Coral `#E55B64`
- **Logo Style:** Hand-drawn/sketched aesthetic
- **Tagline:** "Response to Acute Crisis Training"

#### RASH (Response to Anaphylaxis & Severe Hypersensitivity)
- **Primary Color:** Sky Blue `#5DADE2`
- **Icon:** Epinephrine/adrenaline autoinjector
- **Style:** Friendly, illustrative

#### COACH RED (Defibrillation Mnemonic)
- **Format:** Vertical acronym layout
- **Colors:** Navy headers, color-coded steps
- **Purpose:** Quick reference cognitive aid

### Sub-Brand Hierarchy

All sub-brands should include "powered by REdI" or the REdI logo to maintain brand architecture.

```
[Program Logo]
                    powered by
                    [REdI Logo]
```

## Visual Elements

### Graphic Devices

#### Hexagonal Frames
- Used for photography and image containers
- Suggests interconnected teamwork
- Apply with 2px stroke in brand colors or filled background

#### Color Bars/Stripes
- Diagonal or horizontal gradient stripes
- Use brand colors in sequence: Lime → Teal → Navy
- Apply to document covers and promotional materials

#### Arrow/Chevron Patterns
- Indicate progression and flow
- Use in process diagrams and timelines
- Brand color gradient fills

### Iconography Style

- **Style:** Outlined, 2px stroke weight
- **Corner radius:** Rounded (2px)
- **Colors:** Single color from brand palette
- **Size:** Design at 24x24px, scale proportionally

### Data Visualization

- Use brand color palette
- Primary data: REdI Coral, REdI Navy, REdI Teal
- Secondary data: Light Teal, Lime Green, Sky Blue
- Always include labels and legends
- Maintain accessibility with patterns for colorblind users

## Photography & Imagery

### Photography Style

- **Setting:** Clinical environments, simulation labs, training spaces
- **Subjects:** Multidisciplinary teams, hands-on training, equipment
- **Mood:** Active, collaborative, professional
- **Lighting:** Bright, clean, natural where possible

### Image Treatment

- Apply hexagonal crop for feature images
- Use semi-transparent color overlay for background images:
  - REdI Navy at 60-80% opacity
  - REdI Teal at 40-60% opacity

### Stock Photography Guidelines

Avoid:
- Overly staged or artificial poses
- Non-Australian clinical settings (equipment differences)
- Images that don't reflect workforce diversity
- Outdated equipment or practices

## Accessibility

### Color Contrast

All text must meet WCAG 2.1 Level AA standards:

| Combination | Contrast Ratio | Status |
|-------------|----------------|--------|
| Navy on White | 12.5:1 | ✓ AAA |
| White on Navy | 12.5:1 | ✓ AAA |
| Coral on White | 3.8:1 | ✓ AA Large |
| White on Coral | 3.8:1 | ✓ AA Large |
| Black on Light Teal | 8.2:1 | ✓ AAA |
| Navy on Light Gray | 11.8:1 | ✓ AAA |

### Focus States

All interactive elements must have visible focus indicators:

```css
:focus-visible {
  outline: 3px solid var(--redi-teal);
  outline-offset: 2px;
}
```

### Motion Preferences

Respect user motion preferences:

```css
@media (prefers-reduced-motion: reduce) {
  * {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.01ms !important;
  }
}
```

### Text Requirements

- Minimum body text size: 16px
- Line height minimum: 1.5 for body text
- Maximum line length: 75 characters
- Sufficient spacing between paragraphs
